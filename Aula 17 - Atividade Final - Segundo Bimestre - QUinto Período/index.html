Projeto Web I

Este projeto de Desenvolvimento Web I tem como objetivo desenvolver um site que possa ajudar a sociedade com os problemas encontrados nos processos que englobam o uso do computador. Desde formatação a uma simples funcionalidade de software. Em conjunto a esta, está relacionado o projeto de Gerenciamento de Projetos, com o modelo PM Canvas que compacta a documentação do software e também o projeto de Engenharia de Software II que vem com o objetivo de testar o que é funcional no site.
